Небольшой проект, который я реализовывал в рамках Чемпионата высоких технологий (Профессионалы).
Это небольшие наработки, котрые у меня остались при подготовке к компетенции "Аналитик транспортных данных".
Стек технологий:
1) База данных - SQLite
2) Потоковая обработка - Python + OpenCV
3) Детекция ТС - YOLOv8n (Ultralytics)
4) Пакетная обработка - Pandas / SQL
5) Оркестрация - Apache Airflow (Docker)
6) ML Прогноз - LinearRegression )sklearn)
7) Дашборд - Streamlit + Plotly
8) Инфраструктура - Docker
